Smart way to managing subscription products.

== Description ==

The Smart Subscription For Woocommerce plugin is designed to provide an intelligent subscription management system for WooCommerce, making it easy to handle recurring payments and subscriptions seamlessly.

## Features

- **Intelligent Subscription Management**: Smartly manage subscriptions and recurring payments for your WooCommerce products.
- **Flexible Subscription Options**: Offer various subscription plans, durations, and pricing options tailored to your business needs.
- **Automated Renewals**: Automatically handle subscription renewals, ensuring uninterrupted service for your customers.
- **Customer-Friendly Interface**: Provide a user-friendly interface for customers to manage their subscriptions conveniently.
- **Detailed Reporting**: Gain insights into subscription performance with comprehensive reports and analytics.
- **Integration with WooCommerce**: Seamlessly integrate with WooCommerce, leveraging its powerful e-commerce capabilities.

= Installation =

1. Download the plugin zip file.
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Configure the plugin settings as per your requirements.

## Usage

Once the plugin is activated, you can start setting up subscription plans and products with recurring payment options. Customers can then subscribe to these products and manage their subscriptions through their accounts.

= Setting Up Subscription Plans =

1. Navigate to the WooCommerce settings.
2. Configure subscription settings, such as enable plugin, change label and so on.
3. Save your settings.

= Creating Subscription Products =

1. Create a new product or edit an existing one.
2. Enable the "Subscription" option.
3. Configure subscription details, such as pricing, billing cycle, and expiry.
4. Save or update the product.

== Frequently Asked Questions ==

= How do I create a subscription product? =
Go to product edit page, choose simple product type, enable the checkbox of subscription and fill the necessary details like interval, expiry

= How can I manage recurring payment =
Plugin manages iteslf recurring  revenue, you do not need to worry about this

= Is this plugin compatible with WooCommerce Stripe payment plugin = 
Yes, this is fully compatible with WooCommerce Stripe payment gateway
